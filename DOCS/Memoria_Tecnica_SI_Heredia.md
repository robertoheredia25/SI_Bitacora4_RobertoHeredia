# 1. Análisis de Necesidades
## ¿Qué problema de la empresa resolvemos con Guacamole y Docker

El problema que solucionamos en la empresa gracias a docker es el "en mi ordenador si funciona" ya que dorker lo que hace es meter en una especie de contenedor una aplicacion con todas las cosas que necesita para poder ser ejecutada librerias bases de datos aplicaciones y gracias a esto podemos transportar y usar nuestra aplicacion en cualquier ordenador ya que docker se puede instalar en cualquier sistema operativo ya sea windows , linux y mac.

## ¿Por qué elegimos esta solución y no conectar directamente por RDP a cada máquina?

Usamos docker en vez de una maquina ya que es mas transportable,mas liviano , usa menos recursos ya que no instalas un sistema operativo dentro de un sistema operativo si no que usas los recursos de tu sistema anfitrion esto ayuda a que ordenadores con menos recuros no tengan que tirar para poder usarlo de dos sistemas operativos a la vez lo queh ace que los ordenadores se ralentizen y puedan llegar a apagarse por el sobre fuerzo y nos permite tener en un mismo contenedor varias cosas abiertas como un erp y una base de datos de la que tirar.
